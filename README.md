# Fusion Force | Adding more fusions to Balatro

Requires Fusion Jokers mod (https://github.com/wingedcatgirl/Fusion-Jokers)

Currently contains 55+ fusions using Vanilla Jokers! See the Collection ingame for more information!

Cross-Mod content:
- Ortalab has 10+ fusions! (Also artist credit display for the individual cards)
- CardSleeves (One Sleeve version of the deck)
- Partner with 15+ new choices, and compatibility with base ones
- Tsunami (One gold fusion. Mostly excludes this mod because Splash having so many fusions causes the shop and collection wiggle to go nuts. Shadow spectral also ignores Splash or else it would generate that a LOT. Also Tsunami has a dedicated spectral for Splash already.)

## changelog

V1.3.0

The Partners crossmod update! Partners will now treat fused jokers as their components when it comes to the link bonus. This applies to the base Fusion Jokers mod fusions too (and Tsunami fusions!). Updated several joker descriptions to make them clearer. Serial killer extremely nerfed. Master's Degree no longer makes *enhanced* aces, and now adds seals to scored aces. Soothsayer reported to not work on booster packs even though it's coded to, so updated code (And fixed issue where it was not being triggered by Two-Heads). Fixed Solar Flare Joker using some real jank outdated code that was not properly getting higher probabability benefits. Fixed Midas Touch not giving extra hand size when copied. Altered the randomness of Black Soul. Tweaked Minus Deck and Sleeve to give vouchers instead of money.

Added Fusions: Hybrid Fusion, [Lighthouse and Card Shark, if Tsunami is not already running]

Added Partners: Offer, Shh!, Neigh, Prizes, Suitable, Magnetic, Touch, Shrink, Teamwork, Nice, Lucky Break, Front, Fistbump, Indecision, Forget, Onward, Emote

Added gold fusions: Serial Killer

Known bugs: Onward Partner cannot display a message to show it lost Chips, any attempt to make it do so gives nonsense "card = nil" crashes. Rewards Card probably negates other debt based jokers. Bargaining Chips doesn't show confirmation message of the blind being counted. Card Collection says "+1 Joker" even when it gives +2. Having 3 cards that can all fuse with each to make 3 different fusions bugs out the fusion selection and prevents any from being fused.
